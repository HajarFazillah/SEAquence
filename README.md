## 📌 프로젝트 개요 (Project Overview)

**프로젝트명** <br>Talkativ; 다국적 사용자의 한국어 의사소통을 돕는  AI 기반 맥락 인식 아바타 대화 시뮬레이션 및 코칭 시스템

**프로젝트 내용** <br>Talkativ는 한국에 거주하는 다국적 비원어민 한국어 학습자가 실제 대화 상황에서 겪는
경어법 실수, 말하기 불안, 즉각적 피드백 부재 문제를 해결하기 위해 설계된 AI 기반 모바일 시스템입니다.
사용자는 관계와 상황이 설정된 AI 아바타와 1:1대화 시뮬레이션을 진행하며,
시스템은 자연어처리 (KoNLPy)와 머신러닝 기반 관계.경어법 분석을 통해 사용자의 발화를 실시간으로 분석합니다.

분석 결과를 바탕으로 HYPERCLOVA X가 맥락에 맞는 교정 표현과 원어민 예시 문장을 코칭 피드백으로 지공하며,
세션 종류 후에는 전체 대화에 대한 학습 리포트를 생성합니다. 본 시스템은 단순한 언어 학습 도구가 아닌,
관계.상황.말투를 종합적으로 인식하는 맥락 인식 기반의 한국어 발화 코칭 시스템을 목표로 합니다.

---

## 🎯 프로젝트 목적 (Project Goals)

- 한국어 학습자가 실제 대화 상황에서 느끼는 **언어적·심리적 부담 완화**
- 대화 상대와의 **관계(지위, 나이, 친밀도)**를 고려한 말투·표현 학습 지원
- 단순 번역이나 챗봇이 아닌, **대화 맥락 기반 한국어 코칭 시스템 설계**
- Capstone A 단계에서 **시스템 설계·기술 구조·차별성 정의** 완료

---

## 🧠 핵심 기술 개요 (Core Technologies)

Talkativ는 다음과 같은 핵심 기술 요소를 기반으로 설계되었습니다.

### 1. AI 아바타 대화 시스템
- 역할별 말투, 성격 반영, 문맥 인식
- 한국 LLM ‘HyperCLOVA X’ 기반 응답생성

### 2. 관계·경어법 분석 엔진 
- KoNLPy 형태소 분석으로 정확한 어미/조사 추출
- 역할 기반 위계 관계 분석
- 관계에 따른 말투 적절성 검사

### 3. 실시간 오류 감지 및 교정
- 한국어 문장 교정 및 원어민 표현 비교
- HyperCLOVA X 기반 상황별 어휘·표현 추천

### 4. 개인화 학습 관리 
- 세션별 기록 및 점수 추적
- 실수 패턴 분석 및 학습 진도 관리
